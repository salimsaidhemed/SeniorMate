# SeniorMate
SeniorMate is a modern caregiver and patient management platform designed for home healthcare agencies, adult day care centers, assisted living facilities, and independent caregivers.

The platform provides a centralized system for managing patients, caregiver visits, nursing assessments, medical records, care plans, and clinical documentation. SeniorMate aims to replace paper-based workflows with secure digital records while remaining simple to deploy and operate.

Built using Flask, Vue.js, PostgreSQL, Keycloak, and MinIO, SeniorMate is designed as a practical, production-ready healthcare application that can evolve from a small clinic deployment to a larger enterprise environment.

## Core Features

* Patient management and demographics
* Caregiver and nursing visit tracking
* Digital Aide Notes and Nursing Progress Notes
* Medical records and document management
* Patient assessments and care plans
* Dashboard and operational analytics
* Role-based access control
* Printable clinical documentation
* Secure file storage
* Docker and Kubernetes deployment support

## Technology Stack

* Backend: Flask
* Frontend: Vue 3 + Vuetify
* Database: PostgreSQL
* Authentication: Keycloak
* File Storage: MinIO
* Deployment: Docker Compose / Kubernetes

## Vision

SeniorMate aims to become a complete digital care management platform that enables healthcare providers to focus more on patient care and less on administrative overhead.
