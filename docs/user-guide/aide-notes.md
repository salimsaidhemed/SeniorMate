# Aide Notes

Aide Notes document home health aide activity for a specific patient visit.
Each visit can have at most one Aide Note.

## Create an Aide Note

Open a visit and select **Create Aide Note**. The patient and visit are linked
automatically. Complete the required aide name and time fields, then document
the relevant sections:

- Personal care
- Nutrition
- Mental status
- Elimination
- Activity
- Assistive devices
- Housekeeping
- Additional notes
- Signature and staff details

## View and Edit

Saved checklist values are displayed as readable labels rather than raw JSON.
Authorized users can return to edit mode from the note or visit detail page.

## Duplicate Notes

If an Aide Note already exists for the visit, SeniorMate prevents a second one.
Use **View Aide Note** or **Edit Aide Note** instead.

Caregivers, managers, and administrators can manage Aide Notes. Nurses and
viewers have read access under the current role policy.
