# Assessments

Patient Assessments record structured findings and recommendations. An
assessment belongs to a patient and may optionally reference a visit.

## Supported Types

- Fall risk
- Nutrition
- Mobility
- Cognitive
- General

## Create an Assessment

From a patient or visit, select the assessment action and enter:

- Assessment type
- Assessment date
- Performed by
- Status
- Summary
- Findings
- Recommendations

Status defaults to draft and can be changed to completed. When created from a
visit, SeniorMate verifies that the visit belongs to the selected patient.

## Findings

Findings use a flexible structured format so different assessment types can
evolve without a database change. The detail and print views render the saved
content as readable sections.
