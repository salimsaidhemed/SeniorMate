# Dashboard Overview

The dashboard summarizes current SeniorMate activity after sign-in.

![SeniorMate dashboard with demo metrics](../images/dashboard.jpg)

## Metric Cards

The cards show:

- Total patients
- Active patients
- Visits this month
- Aide notes this month
- Nurse notes this month

## Activity Charts

Grouped views summarize:

- Patients by status
- Patients by gender
- Visits by type
- Visits by status

These charts are operational indicators, not clinical decision tools.

## Recent Visits

The recent visits table shows the latest visit date, patient, visit type, staff
role, and status. Use it to move quickly from an overview to a specific visit.

## Empty or Unavailable Data

An empty dashboard means no matching records exist yet. An error message usually
means the API or database is temporarily unavailable; refresh once, then contact
an administrator if it continues.
