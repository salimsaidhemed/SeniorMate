# Printing Reports

SeniorMate provides browser print views for patients, visits, Aide Notes, Nurse
Notes, and assessments.

## Print or Save as PDF

1. Open the relevant detail page.
2. Select the print action.
3. Review the print-friendly view.
4. Select **Print / Save PDF**.
5. Choose a printer or the browser's PDF destination.

Navigation, action buttons, and application chrome are hidden in print output.

## Included Information

- Patient summaries include demographics, emergency contact, diagnosis,
  recent visits, assessments, and medical-record metadata.
- Visit summaries include patient context, timing, staff, notes, linked note
  status, and assessments.
- Care-note and assessment reports present structured data as readable
  sections instead of raw JSON.

Uploaded medical record files are listed but are not embedded in print output.
Always review the preview before printing or saving.
