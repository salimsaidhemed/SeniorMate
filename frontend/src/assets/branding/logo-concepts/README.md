# SeniorMate Logo Concepts

These original SVG concepts are provided for maintainer review. They are not connected to the production application yet.

Open `preview.html` in a browser to compare all concepts on one review board.

## 1. Care Cross Wordmark

![Care Cross Wordmark](01-care-cross-wordmark.svg)

A balanced healthcare cross with a clear horizontal wordmark. This is the recommended default because it remains legible in navigation, login, print, and document contexts.

## 2. SM Path Monogram

![SM Path Monogram](02-sm-path-monogram.svg)

A compact monogram built around a connected path. It is strongest where the mark must stand alone, such as a collapsed sidebar or app icon.

## 3. Care Shield

![Care Shield](03-care-shield.svg)

A protection-oriented symbol for organizations that want a more clinical and governance-focused identity.

## 4. Heart at Home

![Heart at Home](04-heart-at-home.svg)

An abstract home and heart mark that emphasizes home healthcare without using detailed illustration.

## Design Notes

- All concepts use original vector geometry and system-font wordmarks.
- The palette is compatible with the current SeniorMate teal, blue-grey, and neutral UI.
- White backgrounds are included for predictable review; production variants can later add transparent, monochrome, compact, and dark-surface versions.
- Text should be converted to outlines only when a final production logo is selected and exact typography is approved.
- The maintainer may choose any concept or request a new iteration without changing the planned branding architecture.
